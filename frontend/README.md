# ArxivScout Frontend

This is the frontend for ArxivScout, built with Vite + React + Tailwind CSS.

## Setup

1.  Install dependencies:
    ```bash
    npm install
    ```

2.  Run development server:
    ```bash
    npm run dev
    ```

## Features

*   **Dashboard**: Overview of new papers and research activity.
*   **Reports**: Historical research reports with sentence-level paper indexing.
*   **Papers**: Paper management with filtering, dislike feedback, and detailed views.
*   **Profile**: User settings, research focus, and learning mode toggle (Basic vs Innovation).
*   **Heatmap**: Visualization of research activity over time.

## Configuration

Configuration is managed via `.env` file.
